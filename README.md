# Spirited Away Pomodoro Timer

A simple Pomodoro timer inspired by *Spirited Away*.  
The idea was to keep it calm and distraction-free while still feeling atmospheric.

Live site:  
https://asmitamathur18.github.io/Pomodoro-Timer-Spirited-Away-themed-/

---

## Features

- Adjustable work and break durations  
- Start, pause, and reset controls  
- Background music toggle  
- Animated train scene as the background  
- Minimal UI inspired by the film’s tone

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  

Built with vanilla web technologies.
